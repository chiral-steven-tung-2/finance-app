# Finance App

This project helps organize and work with financial statement CSV files.

## Required Folder Structure

Create the following folders in the project root:

```text
statements/
├── chase-bank/
└── chase-freedom/
```

### Folder Purpose

- `statements/chase-bank/`  
  Upload your **bank account CSV files** here.

- `statements/chase-freedom/`  
  Upload your **Chase Freedom credit card CSV files** here.

## Run the Project

```bash
npm install
npm run dev
```
